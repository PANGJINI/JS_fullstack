ReactJs란?
	- 복잡한 사용자 인터페이스를 쉽게 구축하기 위해 페이스북에서 제작된 JS기반 라이브러리
	- 싱글 페이지 애플리케이션이나 모바일 애플리케이션 개발에 주로 사용된다.
	- 특정 데이터에 변화가 생길 때 매번 변화를 주기 위한 코드를 작성하는 것이 아닌, 
	  기존 코드를 없애고 다시 새롭게 렌더링한다.

렌더링 (Rendering)
	- 화면에 보여주는 것
	- 코드가 번역되어 실행된 뒤 화면을 통해 사용자에게 보여진다.
	
초기 렌더링 (Mount)
	- 가장 처음으로 랜더링 되는 것
	- render() 함수를 사용하여 view를 정의하고 초기 렌더링을 진행하게 된다.

리렌더링 (Re-rendering)
	- view를 업데이트 해야할 때 다시 지웠다가 그려야하며, 이를 리렌더링이라고 한다.
	- 전체를 모두 지우고 다시 그리는 것은 비효율적이기 때문에,
	  이전 dom트리와 새로운 dom트리를 비교한 뒤 변경된 요소 부분만 리렌더링된다.

버츄얼돔 (Virtual DOM)
	- 화면보다 가상돔에 먼저 렌더링하여 컴포넌트에 변화가 감지되면, Virtual DOM으로 렌더링되며
	  이전 Virtual DOM과 현재 Virtual DOM을 비교하여 이로 인해 필요한 부분을 리렌더링 시킨다.
